La segunda prueba comienza en la que el usuario va a jugar el juego de escaleras y serpientes, en el cual tirara un dado de 6 caras. En la parte de:

System.out.print("Arroje el dado [1]Si - [0]No: ");
lanzar = entrada.nextInt();

El usuario lanzara el dado y mirara en que lado le salio, avanzando cuadro por cuadro, ademas, si el usuario cae en un cuadro donde hay una escalera asciende y si cae en un cuadro donde hay una serpiente desciende.

Al finalizar el juego el sistema cerrara.

En caso de que el usuario esta jugando y no quiere seguir jugando tiene la opcion de cerrar el juego: [0]No, debe dar en la tecla 0 para cerrar el sistema.
